# مميزات تطبيق أذكاري

* يحتوي على جميع أذكار حصن المسلم

* يحتوي على الرقية الشرعية من القرآن والسنة

* يحتوي علي القرآن الكريم

* تخريج جميع الأذكار

* يحتوي على سبحة

* تصميم بسيط وسهل للإستخدام

* يوجد به الوضع الليلي لراحة العين أثناء القراءة

* لا يحتوي على أي إعلانات

* مساحة التطبيق صغيرة جداً
